# 工作流卡片媒体上传功能

## ✨ 新功能概述

现在二维工作流画布上的卡片支持上传图片和视频！所有媒体文件都会自动上传到 Vercel Blob 存储。

## 🎯 功能特性

### 1. 支持的媒体类型

- **图片**: JPEG, PNG, GIF, WebP 等所有图片格式
- **视频**: MP4, WebM, MOV 等所有视频格式

### 2. 文件大小限制

- **图片**: 最大 5MB
- **视频**: 最大 50MB

### 3. 存储位置

所有媒体文件上传到 **Vercel Blob** 存储，不占用 Neon 数据库空间。

## 📖 使用指南

### 创建新卡片

1. 点击左上角的 **+** 按钮创建新卡片
2. 卡片会自动进入编辑模式

### 上传媒体

在编辑模式下：

1. **输入卡片标题**（必填）
2. **输入卡片描述**（可选）
3. **上传媒体**（可选）：
   - 点击 **🖼️ 上传图片** 按钮选择图片
   - 或点击 **🎬 上传视频** 按钮选择视频
4. 等待上传完成（会显示"上传中..."）
5. 点击 **完成** 按钮保存

### 预览媒体

- **编辑模式**: 媒体会显示在上传按钮上方
- **显示模式**: 媒体会显示在卡片标题下方

### 删除媒体

在编辑模式下：

1. 双击卡片进入编辑模式
2. 点击媒体预览右上角的 **🗑️** 按钮
3. 媒体会从 Blob 存储中删除

### 编辑卡片

- **双击卡片** 进入编辑模式
- 可以修改标题、描述，或更换媒体
- 点击 **完成** 保存更改

## 🎨 界面优化

### 编辑模式

- 顶部蓝色拖动手柄（⋮⋮ 拖动移动）

### 显示模式

- 顶部装饰条（选中时为蓝色）

- **媒体预览**（新增）：
  - 图片：自动适应宽度，最大高度 180px

## 🔧 技术实现

### 数据结构

```typescript
interface Node {
  id: string
  label: string
  description: string
  x: number
  y: number
  width: number
  height: number
  isEditing: boolean
  imageUrl?: string        // 图片 URL（新增）
  videoUrl?: string        // 视频 URL（新增）
  mediaType?: 'image' | 'video' | null  // 媒体类型（新增）
}
```

### API 端点

- **上传**: `POST /api/upload`
  - 接受 FormData（file, nodeId, type）
  - 返回 { url, pathname, contentType, size, mediaType }

- **删除**: `DELETE /api/upload/delete`
  - 接受 JSON { url }
  - 从 Blob 存储中删除文件

### 文件组织

Blob 中的文件按以下结构存储：

```
nodes/
  {nodeId}/
    image-{timestamp}.jpg
    video-{timestamp}.mp4
```

## ⚠️ 注意事项

### 文件大小

- 图片建议压缩后上传，保持在 1-2MB 以内
- 视频建议使用 H.264 编码，保持在 10-20MB 以内
- 超过限制的文件会被拒绝上传

### 浏览器兼容性

- 图片预览：所有现代浏览器
- 视频播放：需要浏览器支持 HTML5 video 标签
- 推荐使用 Chrome、Firefox、Safari、Edge 最新版本

### 性能优化

- 图片会自动适应卡片宽度
- 视频使用原生播放器，不会自动播放
- 大文件上传时会显示"上传中..."状态
