# AI预览模态框移除新建功能

## 更改概述
根据需求，从AIPreviewModal组件中移除了新建节点和新建边的功能，简化用户界面。

## 移除的功能

### 1. 新建按钮
移除了编辑区域顶部的"➕ 新建节点/边"按钮。

**之前的位置：**
- 位于"节点编辑"和"边编辑"切换按钮的右侧
- 根据当前选中的编辑器类型显示不同的文本

### 2. 状态管理
移除了以下状态：
```typescript
const [isCreatingNode, setIsCreatingNode] = useState(false)
const [isCreatingEdge, setIsCreatingEdge] = useState(false)
```

### 3. 模态框组件
完全移除了以下两个组件：

#### CreateNodeModal
- 用于创建新节点的模态框
- 包含节点名称和描述输入框
- 已不再需要

#### CreateEdgeModal
- 用于创建新边的模态框
- 包含起始节点、目标节点和边标签选择器
- 已不再需要

### 4. 模态框渲染
移除了条件渲染代码：
```typescript
{isCreatingNode && (
  <CreateNodeModal ... />
)}

{isCreatingEdge && (
  <CreateEdgeModal ... />
)}
```

## 保留的功能

用户仍然可以：
1. 查看AI生成的节点和边
2. 编辑现有节点的名称和描述
3. 编辑现有边的关系类型
4. 搜索和筛选节点/边
5. 处理重复节点和冲突
6. 保存图谱

## 用户体验影响

### 优点
1. **界面更简洁**：移除了不常用的功能，减少视觉干扰
2. **流程更清晰**：用户专注于查看和编辑AI生成的内容
3. **减少误操作**：避免用户意外创建不必要的节点或边

### 使用场景
- AI预览模态框的主要目的是查看和编辑AI生成的图谱
- 如果用户需要手动创建节点和边，应该在3D编辑页面进行
- 这样的分工更加明确：AI生成 → 预览编辑 → 保存 → 3D页面进一步编辑
