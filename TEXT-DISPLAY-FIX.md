# 节点名称显示修复

## 🐛 问题

小球上没有显示名称

## ✅ 解决方案

### 优化 Text 组件配置

```typescript
<Text
  position={[0, 0, 1.8]}        // 移到球体前方
  fontSize={0.6}                 // 增大字体（从 0.4 到 0.6）
  color="white"                  // 白色文字
  anchorX="center"               // 水平居中
  anchorY="middle"               // 垂直居中
  outlineWidth={0.05}            // 增大描边（从 0.03 到 0.05）
  outlineColor="#000000"         // 黑色描边
  outlineOpacity={1}             // 描边完全不透明
  maxWidth={4}                   // 增大最大宽度
  textAlign="center"             // 文字居中
  depthOffset={-1}               // 确保文字在最前面
>
  {node.name || '未命名'}
</Text>
```

### 关键改进

1. **位置调整**: `position={[0, 0, 1.8]}`
   - 将文字移到球体前方
   - Z 轴偏移 1.8 单位
   - 确保文字不被球体遮挡

2. **字体增大**: `fontSize={0.6}`
   - 从 0.4 增大到 0.6
   - 更容易看清

3. **描边增强**: `outlineWidth={0.05}`
   - 从 0.03 增大到 0.05
   - 黑色描边更明显
   - 白色文字在任何背景下都清晰

4. **深度优先**: `depthOffset={-1}`
   - 确保文字渲染在最前面
   - 不会被其他物体遮挡

5. **最大宽度**: `maxWidth={4}`
   - 从 3 增大到 4
   - 长名称也能完整显示

## 💡 文字显示原理

### Text 组件工作方式

1. **加载字体**: 首次使用时下载字体文件
2. **生成纹理**: 将文字转换为 3D 纹理
3. **渲染到场景**: 作为 3D 对象显示

### 为什么使用描边？

```
白色文字 + 黑色描边 = 在任何背景下都清晰
```

- 浅色背景：黑色描边提供对比
- 深色背景：白色文字清晰可见

### 位置计算

```
球体半径: 1.5
文字 Z 偏移: 1.8
总距离: 1.5 + 1.8 = 3.3 单位
```

文字在球体前方 1.8 单位，不会被遮挡。
