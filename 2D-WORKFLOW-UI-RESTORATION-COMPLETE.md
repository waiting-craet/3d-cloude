## 技术细节

### 新增功能
1. **ResizeObserver 集成**
   - 跟踪节点 DOM 元素的实际尺寸
   - 自动更新连接点位置
   - 支持动态内容调整

2. **连接点计算优化**
   - 使用实际渲染尺寸而非默认尺寸
   - 缓存连接点位置提高性能
   - 节点尺寸变化时自动更新

3. **媒体处理**
   - 支持图片和视频上传
   - 保持媒体宽高比
   - 媒体预览和删除
   - 上传状态管理

4. **连接线交互**
   - 双击编辑标签
   - 悬停高亮效果
   - 宽触摸区域
   - 标签输入框

5. **节点操作**
   - 复制节点功能
   - 删除节点功能
   - 悬停显示操作按钮
   - 编辑模式切换
