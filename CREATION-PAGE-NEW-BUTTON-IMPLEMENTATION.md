# Creation 页面新建按钮功能实现

## 概述

为 creation 页面的"新建"按钮添加了弹窗功能，用户可以创建新项目和图谱，数据保存到 Neon 数据库。

## 实现的功能

### 1. 对话框集成
- 点击左侧导航栏的"新建"按钮打开 CreateProjectModal 对话框
- 复用了 graph 页面的 CreateProjectModal 组件
- 对话框具有半透明背景和模糊效果

### 2. 创建模式
- **新建项目模式**：创建新项目并在其中创建第一个图谱
- **选择现有项目模式**：在已有项目中添加新图谱

### 3. API 路由

#### `/api/projects` (GET)
- 获取所有项目列表
- 按更新时间倒序排列
- 返回项目基本信息（ID、名称、描述、节点数、边数、时间戳）

#### `/api/projects` (POST)
- 创建新项目和图谱
- 使用事务确保数据一致性
- 请求体：
  ```json
  {
    "name": "项目名称",
    "graphName": "图谱名称"
  }
  ```

#### `/api/graphs` (POST)
- 在现有项目中创建图谱
- 验证项目是否存在
- 请求体：
  ```json
  {
    "projectId": "项目ID",
    "name": "图谱名称"
  }
  ```

### 4. 表单验证
- 项目名称不能为空
- 图谱名称不能为空
- 选择现有项目时必须选择一个项目
- 显示友好的错误提示

### 5. 用户体验
- 创建中显示加载状态
- 防止重复提交
- 创建成功后自动关闭对话框
- 创建失败显示错误信息

## 数据库操作

### Project 表
- 创建新项目记录
- 自动生成唯一 ID (cuid)
- 记录创建和更新时间戳

### Graph 表
- 创建新图谱记录
- 关联到项目 (projectId)
- 自动生成唯一 ID (cuid)
- 记录创建和更新时间戳

## 使用方法

1. 访问 `/creation` 页面
2. 点击左侧导航栏的"新建"按钮
3. 在弹出的对话框中：
   - 选择"新建项目"或"选择现有项目"
   - 输入项目名称（新建项目模式）或选择项目（现有项目模式）
   - 输入图谱名称
   - 点击"创建"按钮
4. 创建成功后对话框自动关闭
