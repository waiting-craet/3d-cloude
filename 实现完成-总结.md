## 🗄️ 数据库表（Neon PostgreSQL）

### 1. Project 表
存储项目信息，每个项目有唯一ID
```sql
CREATE TABLE Project (
  id TEXT PRIMARY KEY,
  name TEXT NOT NULL,
  description TEXT,
  nodeCount INT DEFAULT 0,
  edgeCount INT DEFAULT 0,
  createdAt TIMESTAMP DEFAULT NOW(),
  updatedAt TIMESTAMP DEFAULT NOW()
);
```

### 2. Graph 表
存储图谱信息，关联到项目
```sql
CREATE TABLE Graph (
  id TEXT PRIMARY KEY,
  name TEXT NOT NULL,
  description TEXT,
  projectId TEXT REFERENCES Project(id) ON DELETE CASCADE,
  isPublic BOOLEAN DEFAULT FALSE,
  nodeCount INT DEFAULT 0,
  edgeCount INT DEFAULT 0,
  createdAt TIMESTAMP DEFAULT NOW(),
  updatedAt TIMESTAMP DEFAULT NOW()
);
```

### 3. Node 表
存储节点信息，同时关联项目和图谱
```sql
CREATE TABLE Node (
  id TEXT PRIMARY KEY,
  name TEXT NOT NULL,
  type TEXT NOT NULL,
  description TEXT,
  content TEXT,
  x FLOAT DEFAULT 0,
  y FLOAT DEFAULT 0,
  z FLOAT DEFAULT 0,
  color TEXT DEFAULT '#6BB6FF',
  size FLOAT DEFAULT 1.5,
  imageUrl TEXT,      -- 图片URL
  iconUrl TEXT,       -- 图标URL
  coverUrl TEXT,      -- 封面/视频URL
  projectId TEXT REFERENCES Project(id) ON DELETE CASCADE,
  graphId TEXT REFERENCES Graph(id) ON DELETE CASCADE,
  createdAt TIMESTAMP DEFAULT NOW(),
  updatedAt TIMESTAMP DEFAULT NOW()
);
```

### 4. Edge 表
存储节点关系，同时关联项目和图谱
```sql
CREATE TABLE Edge (
  id TEXT PRIMARY KEY,
  fromNodeId TEXT REFERENCES Node(id) ON DELETE CASCADE,
  toNodeId TEXT REFERENCES Node(id) ON DELETE CASCADE,
  label TEXT NOT NULL,
  weight FLOAT DEFAULT 1.0,
  color TEXT,
  style TEXT,
  projectId TEXT REFERENCES Project(id) ON DELETE CASCADE,
  graphId TEXT REFERENCES Graph(id) ON DELETE CASCADE,
  createdAt TIMESTAMP DEFAULT NOW()
);
```

## 🎯 核心功能

### 1. 数据关联
- 项目 → 图谱 (一对多)
- 图谱 → 节点 (一对多)
- 图谱 → 边 (一对多)
- 节点 ↔ 边 (多对多)

### 2. 数据隔离
- 每个项目的数据完全独立
- 查询时自动过滤，只返回当前项目/图谱的数据
- 不同项目的节点和边互不干扰

### 3. 媒体支持
- `imageUrl`: 节点的图片URL
- `iconUrl`: 节点的图标URL
- `coverUrl`: 节点的封面或视频URL

### 4. 级联删除
- 删除项目 → 自动删除所有图谱、节点、边、媒体文件
- 删除图谱 → 自动删除所有节点、边、媒体文件
- 保证数据一致性

### 5. 2D/3D兼容
- 节点存储 x, y, z 坐标
- 2D视图使用 x, y
- 3D视图使用 x, y, z
- 同一数据源，不同视图

## 💡 使用示例

### 创建项目和图谱
```javascript
// 1. 创建项目
const project = await fetch('/api/projects', {
  method: 'POST',
  body: JSON.stringify({
    name: '委屈',
    description: '我的第一个项目'
  })
}).then(r => r.json())

// 2. 创建图谱
const graph = await fetch(`/api/projects/${project.id}/graphs`, {
  method: 'POST',
  body: JSON.stringify({
    name: '知识图谱A',
    description: '核心知识图谱'
  })
}).then(r => r.json())

// 3. 创建节点
const node = await fetch(`/api/graphs/${graph.id}/nodes`, {
  method: 'POST',
  body: JSON.stringify({
    name: '核心概念1',
    type: 'concept',
    x: 0, y: 0, z: 0,
    imageUrl: 'https://blob.../image.jpg'
  })
}).then(r => r.json())
```

### 在视图中显示
```javascript
// 获取当前项目的所有节点
const nodes = await fetch(`/api/projects/${projectId}/nodes`)
  .then(r => r.json())
  .then(d => d.nodes)

// 在2D视图中渲染
nodes.forEach(node => {
  drawNode2D(node.x, node.y, node.color, node.name)
  if (node.imageUrl) {
    drawImage(node.imageUrl, node.x, node.y)
  }
})

// 在3D视图中渲染
nodes.forEach(node => {
  drawNode3D(node.x, node.y, node.z, node.color, node.name)
  if (node.imageUrl) {
    drawImage3D(node.imageUrl, node.x, node.y, node.z)
  }
})
```
