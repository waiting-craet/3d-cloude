# 顶部横幅图片设置指南

## 📸 图片位置

首页顶部的横幅图片应该放在以下位置：
```
public/images/banner.png
```

## 🎯 图片要求

- **文件名**: `banner.png`（或其他支持的格式：jpg, webp 等）
- **推荐尺寸**: 1920 x 320 像素（或任何宽高比，会自动适配）
- **文件大小**: 建议 < 500KB，以保证加载速度
- **格式**: PNG、JPG、WebP 等

## 📝 操作步骤

1. 将你的图片文件复制到 `public/images/` 目录
2. 确保文件名为 `banner.png`
3. 刷新浏览器，图片会自动显示在首页顶部

## 🔄 更换图片

如果要更换图片，只需：
1. 删除旧的 `public/images/banner.png`
2. 放入新的图片文件，命名为 `banner.png`
3. 刷新浏览器

## 💡 图片显示效果

- 图片会自动填充整个横幅区域（320px 高）
- 图片会自动居中显示
- 图片上方有一个半透明的紫色渐变遮罩，确保文字清晰可读
- 图片会自动响应式缩放
