# 树状布局结构

## 概述
知识图谱现在采用树状结构，从左向右呈放射状展开。

## 节点布局

### 根节点（左侧）
- **RAG 系统** (x=-15, y=0, z=0)
  - 大小: 2.0
  - 颜色: #6BB6FF
  - 位置: 最左侧中心

### 分支 1 - 检索路径（上方）
1. **检索模块** (x=-5, y=6, z=0)
   - 大小: 1.5
   - 连接: 根节点 → 检索模块

2. **向量数据库** (x=5, y=8, z=0)
   - 大小: 1.2
   - 连接: 检索模块 → 向量数据库

### 分支 2 - 知识库路径（中间）
1. **知识库** (x=-5, y=0, z=0)
   - 大小: 1.5
   - 连接: 根节点 → 知识库

2. **嵌入模型** (x=0, y=0, z=0)
   - 大小: 1.2
   - 连接: 知识库 → 嵌入模型

3. **文档处理** (x=5, y=0, z=0)
   - 大小: 1.2
   - 连接: 嵌入模型 → 文档处理

### 分支 3 - 生成路径（下方）
1. **生成模块** (x=-5, y=-6, z=0)
   - 大小: 1.5
   - 连接: 根节点 → 生成模块

2. **LLM** (x=5, y=-8, z=0)
   - 大小: 1.2
   - 连接: 生成模块 → LLM

## 结构特点

- **总节点数**: 8个
- **总边数**: 7条
- **布局方向**: 从左向右（x轴）
- **分支分布**: 上中下三个分支（y轴）
- **深度**: 所有节点在同一平面（z=0）
