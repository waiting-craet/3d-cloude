# Icon Files Required

This directory should contain the following custom Morandi-style icon files:

## Required Icons

1. **excel-morandi.svg** - Icon for Excel file type
   - Size: 48x48px recommended
   - Format: SVG preferred (or PNG)
   - Style: Morandi green aesthetic

2. **csv-morandi.svg** - Icon for CSV file type
   - Size: 48x48px recommended
   - Format: SVG preferred (or PNG)
   - Style: Morandi green aesthetic

3. **json-morandi.svg** - Icon for JSON file type
   - Size: 48x48px recommended
   - Format: SVG preferred (or PNG)
   - Style: Morandi green aesthetic

4. **download-morandi.svg** - Icon for download functionality
   - Size: 20x20px recommended
   - Format: SVG preferred (or PNG)
   - Style: Morandi green aesthetic

## Usage

These icons will be integrated into the import page (`/app/import/page.tsx`) to:
- Display file type icons in the file selection cards
- Display download icons on template download buttons

## Style Guidelines

- Use muted, elegant Morandi green tones
- Apply subtle opacity (0.8) for ink wash effect
- Ensure icons are simple and recognizable
- Match the overall aesthetic of the import page redesign
