# 筛选栏逻辑更新 - 单选模式

## 变更说明
筛选栏已从**多选模式**改为**单选模式**。

## 新的筛选行为

### 默认状态
- 首页显示所有图谱（3D、2D、模板混合显示）

### 选择筛选项
- 点击"3D 图谱"按钮 → 只显示 3D 图谱
- 点击"2D 图谱"按钮 → 只显示 2D 图谱
- 点击"热门模板"按钮 → 只显示模板

### 取消筛选
- 再次点击已选中的筛选项 → 取消筛选，显示所有图谱
- 或点击"清除筛选"按钮 → 取消所有筛选

## 实现细节

### 修改的文件

#### 1. `components/gallery/FilterBar.tsx`
**变更**: `handleFilterToggle` 函数

**旧逻辑** (多选):
```typescript
if (filters.includes(filter)) {
  newFilters = filters.filter(f => f !== filter)  // 移除
} else {
  newFilters = [...filters, filter]  // 添加
}
```

**新逻辑** (单选):
```typescript
if (filters.includes(filter)) {
  newFilters = []  // 取消筛选
} else {
  newFilters = [filter]  // 只选择一个
}
```

#### 2. `app/page.tsx`
**变更**: `handleFilterChange` 函数签名

**旧逻辑**:
```typescript
const handleFilterChange = (filter: string) => {
  // 处理单个 filter 字符串
}
```

**新逻辑**:
```typescript
const handleFilterChange = (filters: string[]) => {
  setActiveFilters(filters)  // 直接设置筛选数组
}
```

## UI 变化

### 筛选按钮状态
- **未选中**: 灰色背景，点击可选中
- **已选中**: 蓝色背景和边框，点击可取消
- **悬停**: 背景颜色变深

### 清除筛选按钮
- 仅在有筛选项时显示
- 点击可一键清除所有筛选

### 活跃筛选指示器
- 显示当前已应用的筛选数量
- 单选模式下最多显示 1 个

## 使用示例

### 场景 1: 查看所有图谱
1. 首页加载 → 显示所有图谱
2. 筛选栏显示: 3D 图谱 | 2D 图谱 | 热门模板（都未选中）

### 场景 2: 只看 3D 图谱
1. 点击"3D 图谱"按钮
2. 筛选栏显示: **3D 图谱** (已选中) | 2D 图谱 | 热门模板
3. 广场只显示 3D 图谱

### 场景 3: 切换到 2D 图谱
1. 点击"2D 图谱"按钮
2. 筛选栏显示: 3D 图谱 | **2D 图谱** (已选中) | 热门模板
3. 广场只显示 2D 图谱（自动从 3D 切换）

### 场景 4: 回到显示所有
1. 点击已选中的"2D 图谱"按钮
2. 或点击"清除筛选"按钮
3. 筛选栏显示: 3D 图谱 | 2D 图谱 | 热门模板（都未选中）
4. 广场显示所有图谱

## 技术优势

- **简洁直观**: 用户一次只能选择一个类别
- **易于理解**: 行为符合常见的单选模式
- **性能优化**: 减少 API 调用（单个筛选 vs 多个筛选组合）
- **URL 管理**: 查询参数更简洁（`?filter=3d` vs `?filter=3d,2d`）

## 向后兼容性

- 旧的多选 URL 参数（如 `?filter=3d,2d`）会被自动处理
- 系统会取第一个筛选项作为活跃筛选
- 用户可以手动调整或使用新的单选界面
