# 节点间距最终设定：30单位

## 概述
将节点之间的最小距离设定为30单位，这是一个紧凑的配置，适合需要在一个视图中显示大量节点的场景。

## 最终配置参数

### 统一配置

| 参数 | 值 | 说明 |
|------|-----|------|
| minDistance | 30 | 节点之间的最小距离 |
| springLength | 30 | 连接节点的理想距离 |
| springStrength | 0.08 | 弹簧力强度 |
| repulsionStrength | 1500 | 排斥力强度（降低以适应更小距离） |
| damping | 0.85 | 阻尼系数 |
| iterations | 80 | 迭代次数（减少以提高性能） |

## 参数调整说明

### 为什么降低排斥力？
- 最小距离从100降到30（减少70%）
- 排斥力从5000降到1500（减少70%）
- 保持力的平衡，避免节点过度分散

### 为什么减少迭代次数？
- 从100次减少到80次
- 更小的距离更快达到稳定
- 提高布局计算性能

### 为什么保持弹簧强度？
- 弹簧强度保持0.08
- 确保连接的节点保持合理的吸引力
- 避免连接节点距离过远

## 技术细节

### 力的平衡
```
排斥力 = 1500 / distance²
弹簧力 = 0.08 × (distance - 30)
最小距离强制 = 30单位
```

### 收敛速度
- 更小的距离
- 更快的收敛
- 80次迭代通常足够

### 参数关系
```
repulsionStrength ≈ 50 × minDistance
springLength = minDistance
iterations = 80 (优化性能)
```
