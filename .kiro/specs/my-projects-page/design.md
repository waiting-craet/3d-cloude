# 我的项目页面 - 设计文档

## 概述

"我的项目"页面是用户管理知识图谱项目的核心界面。页面采用现代的卡片网格布局，采用蓝白色主色调。第一个卡片为"新建项目"卡片（新设计：标签 + 虚线框 + 加号图标），其余卡片显示已创建的项目。点击"新建项目"卡片时会弹出创建项目的对话框。

## 组件和接口

### 1. 项目网格容器 (ProjectGrid)

**位置**: 主内容区域

**特性**:
- 响应式网格布局
- 自动换行
- 适当的间距

**子组件**:
- 新建项目卡片
- 项目卡片（多个）

### 2. 新建项目卡片 (NewProjectCard)

**尺寸**: 与项目卡片相同

**布局**:
- 顶部标签: "新建"（黑色文本，左对齐）
- 主体区域: 虚线框 + 加号图标（居中）

**样式**:
- 背景: 白色 (#FFFFFF)
- 边框: 虚线，蓝色 (#5B8DEE)，2px
- 圆角: 12px
- 标签文本: 黑色，14px，600 weight
- 加号图标: 蓝色 (#5B8DEE)，48px
- 阴影: 0 2px 8px rgba(0, 0, 0, 0.04)
- 悬停效果: 背景浅蓝色 (#F0F5FF)，阴影增强

**交互**:
- 点击时打开创建对话框

### 3. 项目卡片 (ProjectCard)

**尺寸**: 固定宽度（约 280px）

**内容**:
- 项目缩略图/占位符
- 项目名称
- 项目描述（截断）
- 创建日期
- 项目状态指示器

**样式**:
- 背景: 白色 (#FFFFFF)
- 边框: 1px #E8EAED
- 圆角: 12px
- 阴影: 0 2px 8px rgba(0, 0, 0, 0.04)
- 悬停: 阴影增强 (0 8px 16px rgba(0, 0, 0, 0.08))，边框蓝色 (#5B8DEE)
- 缩略图背景: 浅蓝色渐变 (#F0F5FF → #E8EEFF)
- 标题: 黑色，16px，600 weight
- 描述: 灰色 (#666666)，13px
- 日期: 灰色 (#999999)，12px
- 状态标签: 蓝色背景 (#F0F5FF)，蓝色文本 (#5B8DEE)

**交互**:
- 点击打开项目
- 悬停显示更多选项（编辑、删除等）

### 4. 创建项目对话框 (CreateProjectDialog)

**位置**: 屏幕中央（模态）

**尺寸**: 600px 宽（响应式）

**背景**: 深色半透明遮罩

**内容**:
- 标题: "新建知识图谱"
- 副标题: "创建新项目或在现有项目中添加图谱"
- 选项卡: "新建项目" 和 "选择现有项目"
- 表单字段
- 操作按钮

**样式**:
- 背景: #2A2A2A
- 文本: #FFFFFF
- 边框: 1px #404040
- 圆角: 12px
- 阴影: 0 20px 60px rgba(0, 0, 0, 0.3)

### 5. 对话框选项卡 (DialogTabs)

**样式**:
- 背景: 透明
- 文本: 白色
- 选中: 蓝色底部边框
- 未选中: 灰色文本

### 6. 对话框输入字段 (DialogInput)

**样式**:
- 背景: #3A3A3A
- 边框: 1px #505050
- 焦点边框: 2px #667EEA
- 文本: 白色
- 占位符: 灰色
- 圆角: 8px
- 内边距: 12px 16px

### 7. 对话框按钮

**取消按钮**:
- 背景: #404040
- 文本: 白色
- 悬停: #505050

**创建按钮**:
- 背景: #1ABC9C (绿色)
- 文本: 白色
- 悬停: #16A085

## 正确性属性

### Property 1: 新建项目卡片位置
*对于任何* 项目网格，新建项目卡片应该始终显示在第一个位置。

**验证**: Requirements 2

### Property 2: 对话框选项卡切换
*对于任何* 对话框状态，选项卡切换应该正确显示对应的表单内容。

**验证**: Requirements 6

### Property 3: 对话框关闭
*对于任何* 打开的对话框，点击取消或按 Escape 键应该关闭对话框。

**验证**: Requirements 6

### Property 4: 项目卡片信息完整性
*对于任何* 项目卡片，应该显示项目名称、描述、创建日期和缩略图。

**验证**: Requirements 3

## 错误处理

- 如果项目加载失败，显示错误状态
- 如果创建项目失败，显示错误提示
- 如果没有项目，显示空状态
