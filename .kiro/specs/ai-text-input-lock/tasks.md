# 实施计划：AI文本输入锁定功能

## 概述

在 AI 知识图谱生成页面（app/text-page/page.tsx）中实现文本输入框的条件禁用功能。当用户上传文件后，文本输入框将被自动禁用并显示相应的视觉反馈，防止用户同时使用多个数据输入源。该功能通过 React 的声明式渲染实现，确保状态一致性和良好的用户体验。

- 标记 `*` 的任务为可选任务，可以跳过以加快 MVP 开发
