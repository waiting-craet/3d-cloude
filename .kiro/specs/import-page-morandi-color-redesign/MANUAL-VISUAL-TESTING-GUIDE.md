# 导入页面莫兰蒂色彩重新设计 - 手动视觉回归测试指南

## 测试概述

本指南提供详细的手动视觉回归测试步骤,用于验证导入页面(app/import/page.tsx)的莫兰蒂色彩重新设计是否符合设计要求,并与创建页面(app/create/page.tsx)保持视觉一致性。

## 测试环境准备

### 1. 启动开发服务器

```bash
npm run dev
```

### 2. 打开浏览器

推荐使用以下浏览器进行测试:
- Chrome (最新版本)
- Firefox (最新版本)
- Safari (最新版本)
- Edge (最新版本)

### 3. 准备测试页面

在浏览器中打开以下两个页面,并排显示:
- 导入页面: `http://localhost:3000/import`
- 创建页面: `http://localhost:3000/create`

## 附录: 浏览器开发工具使用提示

### Chrome DevTools

1. 右键点击元素 → "检查"
2. 在 Elements 面板中查看 Styles
3. 使用 Color Picker 验证颜色值
4. 使用 :hov 按钮模拟悬停状态

### Firefox Developer Tools

1. 右键点击元素 → "检查元素"
2. 在 Inspector 面板中查看 Rules
3. 使用颜色选择器验证颜色值
4. 使用 :hov 按钮模拟悬停状态

### Safari Web Inspector

1. 右键点击元素 → "检查元素"
2. 在 Elements 面板中查看 Styles
3. 使用颜色选择器验证颜色值
4. 使用 :hov 按钮模拟悬停状态
