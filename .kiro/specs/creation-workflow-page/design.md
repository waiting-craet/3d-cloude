# 创建工作流页面 - 设计文档

## 概述

创建工作流页面是用户从首页"开始创建"后进入的主要工作区。页面采用简洁的白色主题，左侧竖状导航栏提供功能导航，右侧主内容区根据导航选择动态显示相应内容。

## 架构

```
┌─────────────────────────────────────────────────┐
│                   页面容器                        │
├──────────────┬──────────────────────────────────┤
│              │                                  │
│   左侧导航栏  │        右侧主内容区              │
│   (固定宽度)  │      (自适应宽度)               │
│              │                                  │
│  - 知识图谱   │  根据导航选择显示：             │
│  - 新建项目   │  - 我的项目列表                 │
│  - AI创建     │  - 新建项目表单                 │
│  - 我的项目   │  - AI创建界面                   │
│  - 导入文件   │  - 文件导入界面                 │
│  - AI创作▼    │  - AI创作子菜单                 │
│    ├ AI提示词 │                                │
│    ├ 长文本   │                                │
│    ├ 文档分析 │                                │
│    └ 结构化   │                                │
│              │                                  │
└──────────────┴──────────────────────────────────┘
```

## 组件和接口

### 1. 左侧导航栏组件 (SideNavigation)

**位置**: 页面左侧，固定宽度（建议 240px）

**结构**:
```
┌─────────────────────┐
│   知识图谱          │  (竖排标题)
│   (竖排布局)        │
├─────────────────────┤
│ [新建项目] [AI创建] │  (两个按钮)
├─────────────────────┤
│ ● 我的项目          │  (默认选中)
├─────────────────────┤
│ [导入文件]          │  (按钮)
├─────────────────────┤
│ ▼ AI创作            │  (可展开菜单)
│   ├ AI提示词        │
│   ├ 长文本分析      │
│   ├ 文档分析        │
│   └ 结构化语言分析  │
└─────────────────────┘
```

**子组件**:

#### 1.1 标题区域 (TitleSection)
- 显示"知识图谱"文本
- 竖排布局（字体垂直排列）
- 字体大小: 18-20px
- 颜色: 深灰色 (#333333)
- 内边距: 20px

#### 1.2 按钮组 (ButtonGroup)
- 两个按钮: "新建项目"、"AI创建"
- 按钮样式:
  - 背景色: 白色
  - 边框: 1px 灰色 (#CCCCCC)
  - 文字颜色: 深灰色 (#333333)
  - 悬停效果: 背景色变浅灰 (#F5F5F5)
  - 内边距: 10px 15px
  - 圆角: 4px
  - 间距: 8px

#### 1.3 菜单项 (MenuItem)
- "我的项目"
- 样式:
  - 默认: 白色背景，深灰色文字
  - 选中: 浅灰色背景 (#F0F0F0)，左侧蓝色竖条 (3px, #0066CC)
  - 悬停: 浅灰色背景 (#F5F5F5)
  - 内边距: 12px 15px
  - 高度: 40px

#### 1.4 导入文件按钮 (ImportButton)
- 样式同按钮组
- 宽度: 100%
- 内边距: 10px 15px

#### 1.5 AI创作下拉菜单 (AICreationDropdown)
- 主菜单项: "AI创作"
- 展开/折叠图标: ▼/▶
- 子菜单项:
  - AI提示词
  - 长文本分析
  - 文档分析
  - 结构化语言分析
- 样式:
  - 主菜单项: 同菜单项样式
  - 子菜单项: 
    - 背景色: 白色
    - 文字颜色: 深灰色
    - 左边距: 20px (缩进)
    - 内边距: 10px 15px
    - 高度: 36px
    - 悬停: 浅灰色背景 (#F5F5F5)
    - 选中: 浅蓝色背景 (#E6F2FF)，左侧蓝色竖条

### 2. 主内容区组件 (MainContent)

**位置**: 页面右侧，自适应宽度

**特性**:
- 白色背景
- 内边距: 30px
- 根据导航选择显示不同内容
- 支持平滑过渡动画

**内容区域**:
1. **我的项目**: 项目卡片网格
2. **新建项目**: 项目创建表单
3. **AI创建**: AI创建选项界面
4. **导入文件**: 文件上传区域
5. **AI创作子菜单**: 对应的AI创作界面

## 样式规范

### 颜色方案
- 背景色: #FFFFFF (白色)
- 文字颜色: #333333 (深灰色)
- 边框色: #CCCCCC (浅灰色)
- 悬停背景: #F5F5F5 (极浅灰)
- 选中背景: #F0F0F0 (浅灰)
- 强调色: #0066CC (蓝色)
- 强调浅色: #E6F2FF (浅蓝)

### 字体
- 字体族: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif
- 标题: 18-20px, 600 weight
- 菜单项: 14px, 400 weight
- 按钮: 14px, 500 weight

### 间距
- 导航栏宽度: 240px
- 内容区内边距: 30px
- 菜单项高度: 40px
- 子菜单项高度: 36px
- 按钮间距: 8px

### 交互效果
- 悬停: 背景色变化 + 鼠标指针变为手指
- 点击: 轻微缩放 (0.98x)
- 过渡: 200ms ease-in-out

## 正确性属性

一个属性是一个特征或行为，应该在系统的所有有效执行中保持真实——本质上是对系统应该做什么的正式陈述。属性充当人类可读规范和机器可验证正确性保证之间的桥梁。

### Property 1: 导航项选中状态唯一性
*对于任何* 导航栏状态，最多只有一个菜单项处于选中状态。

**验证**: Requirements 4

### Property 2: 菜单展开状态一致性
*对于任何* AI创作菜单，展开状态应该与其子菜单的可见性一致。

**验证**: Requirements 6

### Property 3: 内容区与导航同步
*对于任何* 导航项选择，主内容区应该显示对应的内容类型。

**验证**: Requirements 4, 6

### Property 4: 导航栏固定位置
*对于任何* 页面滚动操作，导航栏应该保持在视口左侧固定位置。

**验证**: Requirements 1

## 错误处理

- 如果导航项加载失败，显示默认的"我的项目"
- 如果内容加载失败，显示加载错误提示
- 如果菜单展开失败，保持当前状态
