## 使用说明

### 1. 查看删除按钮
1. 刷新页面（Ctrl+Shift+R）
2. 以管理员身份登录
3. 点击"现有图谱"下拉菜单
4. 应该能看到红色垃圾桶图标 🗑️

### 2. 删除图谱
1. 展开任意项目
2. 点击图谱右侧的垃圾桶图标
3. 弹出白色确认对话框
4. 点击"确认删除"
5. 快速完成（不到 1 秒）

### 3. 删除项目
1. 点击项目标题右侧的垃圾桶图标
2. 弹出确认对话框（显示图谱数量）
3. 点击"确认删除"
4. 快速完成

## 技术实现

### 优化 API 端点
```typescript
// GET /api/projects/with-graphs
const projects = await prisma.project.findMany({
  include: {
    graphs: {
      select: {
        id: true,
        name: true,
        _count: {
          select: {
            nodes: true,
            edges: true,
          },
        },
      },
    },
  },
})
```

### 删除流程
```typescript
1. DELETE /api/projects/{id} 或 /api/graphs/{id}
2. 显示成功消息（alert）
3. 关闭对话框（立即）
4. 如果删除当前选中的 → 刷新页面
5. 否则 → GET /api/projects/with-graphs → 更新列表
```
