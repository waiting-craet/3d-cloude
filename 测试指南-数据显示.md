## 🐛 常见问题排查

### 问题1: 选择图谱后3D视图是空的

**检查步骤**:
1. 打开控制台，查看是否有错误
2. 检查是否有"正在加载图谱数据"的日志
3. 检查Network标签，API是否返回200
4. 检查API响应数据是否包含nodes数组

**可能原因**:
- 图谱中确实没有节点（检查数据库）
- API调用失败（检查网络）
- 图谱ID不正确（检查localStorage）

**解决方法**:
```bash
# 重新创建测试数据
npx tsx scripts/create-test-data.ts

# 清理缓存
访问: http://localhost:3000/clear-cache
```

### 问题2: 2D视图是空的

**检查步骤**:
1. 确认在3D视图中已经选择了图谱
2. 检查控制台是否有"2D视图: 加载图谱数据"的日志
3. 检查store中是否有数据

**可能原因**:
- 没有选择图谱就跳转到2D视图
- store中没有数据

**解决方法**:
1. 先回到3D视图（http://localhost:3000）
2. 选择一个图谱
3. 再跳转到2D视图（http://localhost:3000/workflow）

### 问题3: 切换图谱后数据没有更新

**检查步骤**:
1. 查看控制台是否有"当前图谱变化"的日志
2. 检查Network标签是否有新的API调用

**可能原因**:
- useEffect依赖项配置错误
- currentGraph没有正确更新

**解决方法**:
- 刷新页面
- 清理缓存后重试

### 问题4: 看到404错误

**错误示例**:
```
GET /api/graphs/graph-1768294373820/nodes 404
```

**原因**:
- 使用了旧的本地ID格式

**解决方法**:
```
访问: http://localhost:3000/clear-cache
```
