## 如何测试

1. 启动开发服务器：
   ```bash
   npm run dev
   ```

2. 访问首页：`http://localhost:3000`

3. 登录后点击"开始创作"按钮

4. 验证以下内容：
   - 左侧导航栏显示正确（1/6宽度）
   - 3DGraph标题加粗显示
   - 三个按钮颜色正确（蓝、白、紫）
   - 右侧主内容区显示正确（5/6宽度）
   - 搜索框在右上角
   - 两个功能卡片横向排布（红、绿）
   - 筛选栏显示"我的项目"和两个下拉框
   - 所有悬停效果正常工作

## 设计规范遵循

所有UI元素严格按照以下设计规范实现：

- **导航栏宽度**: 16.67% (1/6)
- **主内容区宽度**: 83.33% (5/6)
- **标题字体**: 24px, 700 weight, #1a1a1a
- **按钮高度**: 40px
- **按钮圆角**: 6px
- **卡片尺寸**: 200px × 120px
- **卡片圆角**: 12px
- **筛选栏高度**: 60px
- **下拉框尺寸**: 140px × 36px
- **过渡动画**: 200ms ease-in-out

## 注意事项

- 当前实现仅包含UI设计，不包含功能逻辑
- 按钮点击事件仅输出控制台日志
- 搜索和筛选功能暂未实现
