## 🔍 如果页面有问题

### 检查清单

#### 1. 浏览器要求
- ✅ 使用现代浏览器（Chrome, Firefox, Edge）
- ✅ 启用 JavaScript
- ✅ 启用 WebGL（硬件加速）

#### 2. 浏览器开发者工具（按 F12）
**Console 标签**：
- 查看是否有红色错误信息
- 常见错误：
  - `WebGL not supported` → 需要支持 WebGL 的浏览器
  - `Failed to fetch` → API 请求失败
  - `Cannot read property of undefined` → 数据加载问题

**Network 标签**：
- 刷新页面（F5）
- 查看所有请求的状态码
- 应该看到：
  - `/` → 200
  - `/api/nodes` → 200
  - `/api/edges` → 200
  - `/_next/static/...` → 200

**Elements 标签**：
- 查看 `<canvas>` 元素是否存在
- 应该在 `<div id="canvas-container">` 内

#### 3. 快速测试命令

在浏览器控制台（F12 → Console）输入：

```javascript
// 测试 API
fetch('/api/nodes').then(r => r.json()).then(console.log)

// 测试 Zustand Store
// 需要先导入，但可以通过 React DevTools 查看
```
