# 3D知识图谱编辑页面 - 访问指南

## 🎯 快速访问

### 方式1：直接访问主页面
```
http://localhost:3000
```

这会显示完整的3D编辑页面，包含：
- ✅ 顶部导航栏（项目管理、搜索、主题切换）
- ✅ 中间黑色3D画布
- ✅ 3D节点和连接线
- ✅ 右侧节点详情面板
- ✅ 左下角浮动添加按钮

### 方式2：从创建页面跳转
1. 访问 `http://localhost:3000/creation`
2. 点击"新建图谱"按钮
3. 选择"三维图谱"
4. 输入图谱名称
5. 点击"创建"
6. 自动跳转到3D编辑页面

### 方式3：从2D页面转换
1. 访问 `http://localhost:3000/workflow`（2D编辑页面）
2. 编辑2D图谱
3. 点击"保存并转换为3D"按钮
4. 自动跳转到3D编辑页面

## 🐛 常见问题

### Q1: 3D画布显示为黑色，没有节点
**A:** 
1. 检查浏览器控制台是否有错误
2. 确保数据库中有节点数据
3. 运行 `npm run db:seed` 填充示例数据
4. 刷新页面

### Q2: 节点无法拖拽
**A:**
1. 确保长按至少100ms才能进入拖拽模式
2. 检查浏览器是否支持指针事件
3. 查看控制台是否有错误

### Q3: 相机无法旋转
**A:**
1. 确保鼠标在3D画布区域
2. 检查OrbitControls是否启用
3. 尝试刷新页面

### Q4: 搜索功能不工作
**A:**
1. 确保已选择一个图谱
2. 检查节点名称是否包含搜索关键词
3. 尝试使用不同的搜索词
