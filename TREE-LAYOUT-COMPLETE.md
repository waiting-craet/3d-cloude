## 故障排除

如果 3D 区域不显示：

### 1. 检查浏览器控制台
按 F12 打开开发者工具，查看是否有错误

### 2. 检查调试指示器
左上角应该显示：
```
Canvas Container Loaded
Nodes: 8 | Edges: 7
```

### 3. 检查 WebGL 支持
访问 https://get.webgl.org/ 确认浏览器支持 WebGL

### 4. 硬刷新
按 Ctrl + Shift + R 清除缓存并刷新

### 5. 检查 API
访问 http://127.0.0.1:3000/api/nodes 应该返回 8 个节点的 JSON 数据
