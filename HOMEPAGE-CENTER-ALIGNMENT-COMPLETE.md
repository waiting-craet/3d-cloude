## 技术细节

### Flexbox 居中技术
使用了现代 CSS Flexbox 布局来实现完美居中：

1. **主容器** - `display: flex` + `flex-direction: column`
   - 创建垂直的 Flex 容器
   - 所有子元素按列排列

2. **水平居中** - `align-items: center`
   - 在交叉轴（水平方向）上居中对齐
   - 适用于所有子元素

3. **垂直居中** - `justify-content: center`
   - 在主轴（垂直方向）上居中对齐
   - 确保内容在视口中垂直居中

4. **文本居中** - `text-align: center`
   - 确保文本内容居中
   - 适用于标题和副标题
