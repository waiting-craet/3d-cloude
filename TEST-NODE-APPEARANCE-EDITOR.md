# 测试节点外观编辑器

### 1. 启动开发服务器
```bash
npm run dev
```

### 2. 打开浏览器
访问 http://localhost:3000

### 3. 确保以管理员身份登录
- 如果未登录，点击右上角的登录按钮

## 🐛 常见问题

### 问题1: 节点名称与形状重叠
**解决方案**: 检查 `getTextYPosition()` 函数的计算是否正确

### 问题2: 形状切换后颜色丢失
**解决方案**: 检查 `createGeometry()` 是否正确应用材质

### 问题3: 保存失败
**解决方案**: 
1. 检查浏览器控制台错误信息
2. 检查网络请求是否成功
3. 检查数据库连接是否正常

### 问题4: 箭头形状显示不完整
**解决方案**: 检查 Node 组件中箭头的圆锥头部是否正确渲染
