# 导航栏搜索功能

## 功能概述

在导航栏中添加了智能搜索功能，允许用户快速搜索和定位当前知识图谱中的节点。

## 主要特性

### 1. 图谱关联搜索
- ✅ 只搜索当前选中图谱的节点
- ✅ 未选择图谱时，搜索框禁用并显示提示
- ✅ 搜索框占位符动态显示当前图谱名称

### 2. 实时搜索
- ✅ 输入关键词即时显示匹配结果
- ✅ 按节点名称进行模糊匹配（不区分大小写）
- ✅ 显示匹配节点的数量

### 3. 搜索结果展示
- ✅ 显示节点名称、类型和描述
- ✅ 鼠标悬停高亮效果
- ✅ 最多显示 300px 高度的结果列表（可滚动）
- ✅ 未找到结果时显示友好提示

### 4. 节点定位
- ✅ 点击搜索结果自动选中节点
- ✅ 相机平滑移动到节点位置
- ✅ 使用缓动动画（800ms）提供流畅体验
- ✅ 自动清空搜索框

## 使用方法

### 基本搜索流程

1. **选择知识图谱**
   - 点击"现有图谱"下拉菜单
   - 选择要搜索的项目和图谱

2. **输入搜索关键词**
   - 在搜索框中输入节点名称的关键词
   - 系统会实时显示匹配的节点

3. **选择节点**
   - 点击搜索结果中的任意节点
   - 相机会自动聚焦到该节点
   - 节点详情面板会显示节点信息

### 搜索提示

- 搜索不区分大小写
- 支持部分匹配（例如搜索"用户"可以找到"用户管理"、"用户登录"等）
- 搜索结果按节点在图谱中的顺序显示

## 技术实现

### 搜索逻辑
```typescript
// 只搜索当前图谱的节点
useEffect(() => {
  if (searchQuery.trim() && currentGraph) {
    const results = nodes.filter(node => 
      node.name.toLowerCase().includes(searchQuery.toLowerCase())
    )
    setSearchResults(results)
    setShowResults(true)
  } else {
    setSearchResults([])
    setShowResults(false)
  }
}, [searchQuery, nodes, currentGraph])
```

## 性能优化

1. **防抖处理**
   - 使用 `setTimeout` 延迟关闭搜索结果（200ms）
   - 避免快速点击导致的闪烁

2. **条件渲染**
   - 只在有搜索结果且图谱已选择时渲染下拉框
   - 减少不必要的 DOM 操作

3. **相机动画优化**
   - 使用 `requestAnimationFrame` 实现流畅动画
   - 缓动函数提供自然的加速/减速效果
   - 动画时长控制在 800ms，平衡速度和体验
