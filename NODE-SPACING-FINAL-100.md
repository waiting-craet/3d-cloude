# 节点间距最终设定：100单位

## 概述
经过测试和调整，最终将节点之间的最小距离设定为100单位。这个值在视觉效果和实用性之间取得了良好的平衡。

## 最终配置参数

### 所有文件的统一配置

| 参数 | 值 | 说明 |
|------|-----|------|
| minDistance | 100 | 节点之间的最小距离 |
| springLength | 100 | 连接节点的理想距离 |
| springStrength | 0.08 | 弹簧力强度 |
| repulsionStrength | 5000 | 排斥力强度 |

## 为什么选择100单位？

### 优点
- ✅ **适中的间距**: 既不太挤也不太松散
- ✅ **良好的可读性**: 节点标签清晰可见
- ✅ **合理的连线长度**: 连线既不太短也不太长
- ✅ **适用范围广**: 适合10-100个节点的图谱
- ✅ **性能良好**: 布局计算时间合理
- ✅ **视野友好**: 容易在一个视图中看到多个节点

### 与其他值的对比

| 距离值 | 优点 | 缺点 | 适用场景 |
|--------|------|------|----------|
| 20 | 紧凑，节点多时可见 | 容易重叠，拥挤 | 大量节点（200+） |
| 100 | **平衡，适中** | **无明显缺点** | **通用场景（推荐）** |
| 1000 | 极其宽松，清晰 | 空间太大，难以浏览 | 少量节点（<10） |

## 技术细节

### 力的平衡
```
排斥力 = 5000 / distance²
弹簧力 = 0.08 × (distance - 100)
最小距离强制 = 100单位
```

### 收敛条件
- 迭代100次
- 或系统能量接近0
- 或没有节点违反最小距离

### 参数协调
- 排斥力强度 ∝ 最小距离²
- 弹簧长度 = 最小距离
- 弹簧强度 ∝ 1/最小距离

- minDistance = springLength
- repulsionStrength ≈ 50 × minDistance
- springStrength ≈ 0.08
