## ⚠️ 注意事项

### Neon 数据库
1. **连接限制**：免费计划有并发连接限制
2. **自动休眠**：长时间不活动会休眠（首次查询可能较慢）
3. **备份**：建议定期备份重要数据
4. **SSL 必需**：必须使用 SSL 连接

### Vercel Blob
1. **文件大小限制**：
   - 图片：最大 5MB
   - 视频：最大 50MB
2. **存储配额**：免费计划 1GB
3. **带宽限制**：免费计划 100GB/月
4. **公开访问**：所有文件默认公开（URL 难以猜测）
