# 添加节点功能说明

## ✅ 已完成的功能

### 权限控制 🔒

**重要**: 添加节点功能仅对管理员开放

- ✅ 浮动添加按钮仅在管理员登录后显示
- ✅ 普通用户无法看到添加按钮
- ✅ 管理员登出后，按钮自动隐藏
- ✅ 实时响应登录/登出状态变化

**实现机制**:
- 组件内部检查 `localStorage` 中的 `isAdmin` 状态
- 监听 `loginStateChange` 自定义事件
- 登录/登出时自动显示/隐藏按钮
- 无需刷新页面即可响应状态变化

## 📋 使用流程

### 前提条件

⚠️ **必须以管理员身份登录才能使用添加节点功能**

1. 点击导航栏右上角的"登录"按钮
2. 输入管理员账号：`admin`
3. 输入密码：`admin123`
4. 登录成功后，右下角会出现浮动添加按钮

### 添加节点

1. **点击浮动按钮**
   - 位于右下角的蓝色圆形按钮
   - "+" 图标

2. **填写节点信息**
   - 节点名称（必填）：例如 "人工智能"
   - 节点介绍（选填）：例如 "人工智能是计算机科学的一个分支..."

3. **提交创建**
   - 点击"添加"按钮
   - 或按 Enter 键（在名称输入框中）

4. **节点创建成功**
   - 弹窗关闭
   - 节点出现在 3D 场景中
   - 随机位置和蓝色外观

### 取消操作

- 点击"取消"按钮
- 点击弹窗外部背景
- 按 Esc 键（可选实现）

### 状态管理

```typescript
// FloatingAddButton 状态
const [isModalOpen, setIsModalOpen] = useState(false)
const [isAdmin, setIsAdmin] = useState(false) // 管理员状态

// 权限检查
useEffect(() => {
  const savedIsAdmin = localStorage.getItem('isAdmin')
  setIsAdmin(savedIsAdmin === 'true')

  // 监听登录状态变化
  const handleLoginStateChange = () => {
    const currentIsAdmin = localStorage.getItem('isAdmin')
    setIsAdmin(currentIsAdmin === 'true')
  }
  
  window.addEventListener('loginStateChange', handleLoginStateChange)
  return () => {
    window.removeEventListener('loginStateChange', handleLoginStateChange)
  }
}, [])

// 非管理员不渲染按钮
if (!isAdmin) return null

// AddNodeModal 状态
const [nodeName, setNodeName] = useState('')
const [nodeDescription, setNodeDescription] = useState('')
const [error, setError] = useState('')
```

### 关键函数

```typescript
// 添加节点
handleAddNode = async (name, description) => {
  await addNode({
    name,
    description,
    type: 'entity',
    x: Math.random() * 20 - 10,
    y: Math.random() * 15,
    z: Math.random() * 20 - 10,
    color: '#4A9EFF',
    size: 1.5,
  })
}

// 表单验证
handleSubmit = (e) => {
  e.preventDefault()
  
  if (!nodeName.trim()) {
    setError('请输入节点名称')
    return
  }

  onAdd(nodeName.trim(), nodeDescription.trim())
  // 重置表单...
}
```
