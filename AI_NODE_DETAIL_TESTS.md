# AI 节点详情提取测试用例

根据需求，提供以下 3 组测试文本，用于验证文本解析和知识图谱生成阶段对“节点–详情”映射的正确处理。

## a) 所有节点均含详情

```text
苹果公司是一家美国的跨国科技公司。
史蒂夫·乔布斯是苹果公司的联合创始人之一。
iPhone是苹果公司研发的一款智能手机。
```

## b) 部分节点含详情

```text
特斯拉公司是一家美国的电动汽车及能源公司。
埃隆·马斯克是特斯拉公司的首席执行官。
Model 3
Model S
```
*(注：Model 3 和 Model S 作为节点，在此文本中没有详情段落)*

## c) 所有节点均不含详情

```text
微软
比尔·盖茨
Windows操作系统
```
*(注：所有文本均为独立的实体，不包含描述、定义或属性等详情)*
